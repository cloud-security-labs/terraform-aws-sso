## Assumptions

- AWS ORG Created
- AWS SSO Enabled
- AWS SSO Groups Created

## Requirements

| Name      | Version   |
| --------- | --------- |
| terraform | >= 0.14.0 |
| aws       | >= 3.24.0 |

## Providers

| Name | Version   |
| ---- | --------- |
| aws  | >= 3.24.0 |
